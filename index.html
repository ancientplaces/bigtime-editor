MIDI Settings
Global Notes (CC 112): OFF = MIDI channel only. Channel = respond to channel. Global = respond on any channel.
MIDI Clock CC 110
No Clock: MIDI OUT acts as THRU (default). Clock In: Big Time syncs to incoming MIDI clock. Clock Out: Big Time emits its own clock (60–240 bpm) via MIDI OUT — auto-subdivides if TIME/MODE push outside range. Not available in MOD mode.
EOM — Expression Over MIDI CC 100
EOM allows an expression pedal connected to a MIDI controller to control Big Time's faders over MIDI. Configure the expression assignment in the Exp/CV tab first, then send CC 100 with value 0–127 to control the assigned fader(s).

Expression / CV Assignment
How it works on the pedal: Hold SHIFT, then move any fader to max to assign Exp/CV to it. Right footswitch = set "toe" (max) position. Left footswitch = set "heel" (min) position.

Reference / planning only: Big Time's Exp/CV assignment and heel–toe range are set on the hardware using the steps above — there is no MIDI command to configure them remotely. Use this tab to plan and note your assignments; the values are saved with your local presets. For real-time expression over MIDI, use EOM (CC 100) on the System & MIDI tab. CV range: 0–5V — higher voltage may damage the pedal.
